# DocuFlow Demo 0.1

Demo inicial de DocuFlow para validación regulatoria automatizada de documentos
(SARLAFT / modelos / trazabilidad) con:

- Backend: FastAPI
- Frontend: Streamlit
- Reglas binarias (Golden Rule Set 1.0)
- Evidencia estructurada en JSON
- Generación básica de reporte PDF

## 🚀 Cómo ejecutar localmente

### 1. Crear entorno e instalar dependencias

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
